:::tip

Dersom du har en Windows maskin så husk å trykke på _'Windows(PowerShell)'_ under for å få riktige kommandoer.

:::


For å laste ned disse applikasjonene må du åpne Terminalen på Mac eller Linux og lime inn linjene under. På Windows må du åpne PowerShell.

<Tabs>

<TabItem value="nix" label="Linux/Mac OS" icon="linux" default>

<Steps>

1. **Installer Homebrew** 

   [Homebrew](https://brew.sh/)

   ```sh
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

   ```

2. **Installer OpenCode** 

   [OpenCode](https://opencode.ai/)

   ```sh
   brew install --cask opencode-desktop
   ```

3. **Installer VS Code**

   [VS Code](https://code.visualstudio.com/)

   ```sh
   brew install --cask visual-studio-code
   ```
4. **Installer Node.js**

   [Node.js](https://nodejs.org/en)

   ```sh
   brew install node
   ```

</Steps>

</TabItem>

<TabItem value="windoze" label="Windows (PowerShell)" icon="seti:windows">

<Steps>

1. **Installer WinGet** 

   [WinGet](https://learn.microsoft.com/en-us/windows/package-manager/winget/)

   ```powershell
   Install-Script -Name winget-install 
   ```

2. **Installer OpenCode Desktop** 

   [OpenCode](https://opencode.ai/)

   Follow the link and click download under OpenCode Desktop for windows

   [OpenCode Download](https://opencode.ai/download)


3. **Installer VS Code**

   [VS Code](https://code.visualstudio.com/)

   ```cmd
   winget install -e --id Microsoft.VisualStudioCode
   ```

4. **Installer Node.js**

   [Node.js](https://nodejs.org/en)

   ```cmd
   winget install -e --id OpenJS.NodeJS
   ```

</Steps>

</TabItem>

</Tabs>
