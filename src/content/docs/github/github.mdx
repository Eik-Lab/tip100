### GitHub som en støttespiller

[GitHub til Eik Lab](https://github.com/eik-lab)

Dersom du besøker Eik Lab sin github (trykk på linken ovenfor) så vil du finne alle CAD filer som er blitt brukt under dette kurset.

Dette vil si at hvis det er en liten endring du har gjort som du ikke klarer å finne ut hvor ligger, eller at du har glemt å lagre 
ditt prosjekt, så er det mulig å laste ned CAD-filene fra GitHub repoet.

### Hvordan navigere seg i GitHub

Repoet på GitHub vil se slikt ut:

<FileTree>

- tip100
  - modul1/
    - xxx.f3z
    - xxx.f3z
    - xxx.f3z
  - modul2/
    - xxx.f3z
    - xxx.f3z
    - xxx.f3z
  - utfordring/
    - xxx.f3z
    - xxx.f3z
    - xxx.f3z
  - flake.nix 
  - README.md 

</FileTree>

Alle CAD filer vil være lastet opp som '.f3z' dette er filnavnet som Fusion 360 bruker. Det er disse filene du i så fall ønsker å laste ned på din PC.
Alle filene som er relevante vil du kunne se ovenfor er merket. For å navigere seg til riktig fil er det bare å trykke på mappen som filen befinner seg i.
